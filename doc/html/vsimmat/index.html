vsimmat

Background
The program 'vsimmat' computes a similarity matrix based on correlations between time series. The time course in every voxel is correlated with the time course of every other voxel. Only voxels inside the user-defined mask are used. This matrix can subsequently be analyzed using other programs, e.g. vspectralcluster.

Reference:
Lohmann et al.(2009). ISMRM.

Usage
Here is a typical calling sequence which will produce a clustering of fMRI data within a specified region of interest with 3 clusters. The region of interest can be defined using 'vledit' or some other means.



vsimmat -in data.v -out mat.v -mask roi.v -fisher true

vspectralcluster -in mat.v -out result.v -n 3


Parameters of 'vsimmat':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-mask
Mask defining a region of interest. Required.
-minval
Signal threshold. Default: 0
-fisher
Whether to do Fisher r-to-z transform. Default: false
