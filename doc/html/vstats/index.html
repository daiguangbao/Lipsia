vstats

Background
The program 'vstats' computes various voxelwise statistics across a set of input images. The images may for instance be a set of maps resulting from an ECM analysis.

Usage
Here is a typical calling sequence.

vstats -in data_*.v -out result.v -type skewness

If for example "data_*.v" represents a set of 20 ECM maps, then the above call will yield a map showing the skewness of the 20 values per voxel.

Choices for the parameter 'type' are:

[mean | std | skewness | kurtosis | min | max | median | stderr | numpos | sum | cv]

'std' means 'standard deviation', 'stderr' is standard deviation devided by sqrt(n). 'cv' means 'coefficient of variation', i.e. std/mean.

Parameters of 'vstats':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-type
[mean | std | skewness | kurtosis | min | max | median | stderr | numpos | sum | cv]
Type of statistics to compute. Default: mean
