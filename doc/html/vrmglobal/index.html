vrmglobal

Background
The program 'vrmglobal' can be used as a part of the preprocessing pipeline. It regresses out a global signal component from fMRI data. The global signal component is computed as the mean signal value per timestep where the mean is computed across all voxels in a region of interest.

Usage
Here is a typical calling sequence.



vrmglobal -in data.v -out result.v -mask mask.v -minval 0


Parameters of 'vrmglobal':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-mask
Mask defining the region in which the global signal is to be computed. Required.
-minval
Signal threshold to exclude non-brain voxels. Required.
