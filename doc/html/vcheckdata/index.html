vcheckdata

Background
The program 'vcheckdata' is a tool for checking fMRI data data quality. It computes statistics (mean, standard deviation, kurtosis, etc.) of the timecourse in each voxel. It can be used to check for scanner problems such as inhomogeneities. If the parameter '-high' is set to some value > 0, a detrending is performed prior to the computation of the statistics. The detrending is done using a high-pass filter with cutoff wavelength of seconds.

Usage
The following call computes the standard deviation (std) of each voxel's time course. Prior to the computation of the standard deviation, the data are detrended such that slow drifts with frequencies of less than 1/60 Hz are removed. The input data set remains unchanged.

vcheckdata -in data.v -out result.v -type std -high 60


Parameters of 'vcheckdata':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-type
Type of statistics to compute.
Choices are: [mean | std | percent | skewness | kurtosis | min | max | median]. Default: std
-high
Upper wavelength threshold in seconds for detrending. Default: 60
