LIPSIA     Wilcoxon's rank test - random effects analysis - statistical significance of an effect
vwilcoxon
This program performs a Wilcoxon test for matched pairs on a group of contrast images. The ouput is a zmap that gives the voxel-wise statistical significance of an effect.

Note that each voxel in a contrast image contains a difference between effects. E.g. for two effects b1 and b2, we may have a contrast b1 - b2. Thus, the matched pair in this example consists of b1 and b2.

vwilcoxon -in cgtbsm_*.v -out zmap.v


Parameters of 'vwilcoxon':
-help
Prints usage information.
-in
Input files. Default: (none)
-out
Output file. Default: (none)
