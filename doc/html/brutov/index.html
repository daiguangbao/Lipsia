LIPSIA     Conversion of Bruker data to Vista
brutov2

The program 'brutov2' converts Bruker data to the vista data format. Example:

brutov2 -in brukerdirectory -out bruker.v -exp 2,4,5

Parameters of 'brutov2':
-help
Prints usage information.
-in
source location of the Bruker data to be converted
(the top-level directory of the dataset)
-out
target location of the converted Vista data
-exp
the Bruker experiments to be converted
(e.g. -exp 3, multiple experiments can be given: -exp 2,4,5)
-b
lower histogram cut-off value (percentage of the histogram area) 0.0 ... 100.0
-w
upper histogram cut-off value (percentage of the histogram area) 0.0 ... 100.0
-v [ debug | info | warning | error ]
level of output verbosity.
-p [ auto | original | byte | short ]
precision (bytes per voxel) of the created Vista data. Default: auto
-nrep
the number of repetitions to convert (0 means all repetitions)
-fa
force the program to treat the input data as anatomical
-ff
force the program to treat the input data as functional
-fbo [ big | little ]
force the program to use the given byte order
