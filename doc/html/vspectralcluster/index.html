vspectralcluster

Background
The program 'vspectralcluster' produces a clustering based on a similarity matrix. This matrix must have been produced beforehand by some other program, e.g. 'veta2' or 'vsimmat'. The parameter '-type' specifies which algorithm is to be used: '-type Shi' selects the Shi,Malik variant and '-type Ng' selects the Ng,Jordan,Weiss variant of spectral clustering.

References:
(1) Shi,J and Malik,J (2000), Normalized cuts and image segmentation, IEEE-PAMI, 22(8), pp. 888-905.
(2) Ng AY,Jordan MI, Weiss Y (2002). Proc. Neural Information Processing Systems (NIPS), 14:849-856.

Usage
Here is a typical calling sequence using a correlation based similarity matrix as input constructed by 'vsimmat':

vsimmat -in data.v -out mat.v -mask roi.v

vspectralcluster -in mat.v -out result.v -n 3



Alternatively, a connectivity-based similarity matrix constructed by 'veta2' may be used:

veta2 -in data.v -out mat.v -mask roi.v

vspectralcluster -in mat.v -out result.v -n 3


Parameters of 'vspectralcluster':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-n
Number of clusters. Default: 3
-type
Type of algorithm to use. Default: Ng
