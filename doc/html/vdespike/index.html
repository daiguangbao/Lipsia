vdespike

Background
The program 'vdespike' tries to remove artefacts from time series data. It assume that artefacts result from subject movement so that some time steps withn the time series are not usable. It detects such time steps based on signal variance. Time steps with large standard deviation are assumed to be artefacts. 'vdespikes' first identifies such time points and then replaces them by interpolating with adjacent time points.




Note: vdespike automatically performs a light baseline drift correction to your data. But this is not a substitution of the baseline drift correction performed by vpreprocess

Usage
Here is a typical calling sequence.

vdespike -in data.v -out result.v -alpha 4

This call replaces time steps which are above (mean + 4*sigma) by interpolated values.


Parameters of 'vdespike':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-alpha
Use mean+alpha*sigma as threshold. Default: 3.5
-verbose
Whether to report artefacts. Default: false
