volumeinfo

Background
The program 'volumeinfo' prints out information about regions of interest contained in a mask file.



Usage
Here is a typical calling sequence.



volumeinfo -in mask.v



Parameters of 'volumeinfo':
-help
Prints usage information.
-in
Input file. Default: (none)
