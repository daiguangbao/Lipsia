Shortcuts

Most of the menu entries and buttons have an assigned key for activation. Here is a list of these shortcuts:

<Space> Switch active object volume/clip
<P> Enable/Disable clipping-plane
<B> Enable/Disable clipping-box
<A> Adjust dialog for light, rotation, z-Values
<D> Enable/Disable shadowing
<V> Open current dataset with vlview
<C> Connect to lipsia-server
<E> Export rendered image to file
<S> Save current view to file
<L> Load view from file
<Z> Enable/Disable zmap colorbar
<1> View Left
<2> View Right
<3> View Top
<4> View Bottom
<5> View Anterior
<6> View Posterior
<Esc> Exit vlRender
