LIPSIA     Image Averaging
vave
This program computes a voxel-wise average over a group of input files. Any type of input can be used, e.g. T1-weighted, T2-weighted, etc. The results is a simple average, not a statistical test.

vave -in image_*.v -out ave.v


Parameters of 'vave':
-help
Prints usage information.
-in
Input files. Default: (none)
-out
Output file. Default: (none)
