vReHo

Background
The program 'vReHo' computes regional homogeneity in fMRI time series data. It produces a map showing regional homogeneity in the 27-neighbourhood around each voxel. It uses Kendall's W as a measure of homogeneity.

Reference:
Zang, Y., Jiang, T., Lu, Y., He, Y., and Tian, L. (2004). Regional homogeneity approach to fMRI data analysis. Neuroimage 22, 394-400.

Usage
Here is a typical calling sequence.



vReHo -in data.v -out result.v


Parameters of 'vReHo':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-first
First timestep to use. Default: 2
-length
length of time series to use, '0' to use full length. Default: 0
