LIPSIA     Design file conversion
vspm2lipsia
This program is a utility for converting onset vectors (used in vglm2 or spm2) into lipsia-design files. Input and output files are text files. The lipsia-design file can be further processed using 'vgendesign', 'vcheckdesign' etc.

Example:

vspm2lipsia -in onset.des -tr 2 -out design.des

The design specification of first level designs is specified in 'designformat'.


Parameters of 'vspm2lipsia':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-tr
Repetition time in seconds. Required.
