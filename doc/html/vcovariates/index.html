vcovariates

Background
The program 'vcovariates' can be used as the first step for regressing out nuisance covariates from fMRI data. Such covariates may for instance result from RETROICOR. 'vcovariates' can be used as part of the preprocessing pipeline, and should be used in conjunction with 'vresiduals' and optionally also with 'vcheckdesign'. 'vcovariates' converts the text file into the vista data format so that it can then be used in a subsequent regression step using the program 'vresiduals'. The input into 'vcovariates' is a text file containing one line per time step. Each line should contain the covariates to be regressed out. The output is a vista-file which can be visualized using 'vcheckdesign'.

Usage
Here is a typical calling sequence.



vcovariates -in nuisance.txt -normalize true -tr 2 -out nuisance.v

vcheckdesign -in nuisance.v &

vresiduals -in fmri_data.v -design nuisance.v -out result.v

The first call generates a design file in vista-format. It also normalizes each covariate to zero mean and unit variance (optional). The resulting the design file can be visualized and checked using 'vckeckdesign'. The last call uses the design file to regress out the covariates.

Parameters of 'vcovariates':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-tr
Repetition time in seconds. Required.
-normalize
Whether to normalize each covariate to zero mean, and unit variance. Default: false
