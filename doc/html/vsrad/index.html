vsrad

Background
The program 'vsrad' removes noise from MRI images. It is used in conjuncton with 'vdenoise'. 'vsrad' is a speckle reducing filter. It used here to reduce artefacts which may result from a prior call to 'vdenoise'. 'vdenoise' is a wavelet filter and may produce Gibbs ringing artefacts. Denoising using 'vdenoise' and 'vsrad' has been tested on structural images (T1-weighted,Flash) and diffusion weighted images. In principle, it should also be usable on fMRI data, but it has not yet been tested in this context.

References:
Lohmann et al (2010, in press). Magn Res Med.
Yu and Acton (2002). IEEE Trans Image Proc, 11(11):1260-1270.



Usage
The program 'vsrad' should be called after 'vdenoise'. The number of iterations in 'vsrad' controls the filter strength. Fewer iterations produce a lesser effect. Here is a typical calling sequence.



vdenoise -in data.v -out tmp.v -level 0.5

vsrad -in tmp.v -out result.v -iter 5




Parameters of 'vsrad':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-iter
Number of iterations. Default: 5
