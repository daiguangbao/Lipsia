LIPSIA     Residuals of a general linear regression using precoloring
vresiduals


Background
The program 'vresiduals' performs a general linear regression (GLM) and outputs the residuals of the GLM as output. As input, it requires fMRI raw data which may have been pre-processed (baseline filtered and/or geometrically transformed into the AC/PC space). In addition, it requires a design matrix as generated by the program 'vgendesign'.

The program performs a pre-coloring of the raw data, i.e. it applies a temporal Gaussian smoothing with a user-specified kernel width given by the parameter '-fwhm' (full width at half maximum). The smoothing imposes a temporal autocorrelation that determines the degrees of freedom. The program yields the residuals of the GLM as output.

Reference: Lohmann et al. (2010). Cerebral Cortex 20(6):1286-1292
http://dx.doi.org/doi:10.1093/cercor/bhp190


Usage
vgendesign -in design.txt -out design.v -tr 2 -ntimesteps 648 -deriv 0

vresiduals -in ntbsm_data.v -design design.v -out residuals.v

This sequence of program calls first generates a design matrix from a text file containing the design specification. It then performs a general linear regression and outputs the residuals.


The parameter '-minval' can be used to exclude voxels (e.g. voxels outside the brain) from the analysis. Voxels whose first value in their time series is below 'minval' will be excluded from processing.

Parameters of 'vresiduals ':
-help
Prints usage information.
-in
One or more fMRT data file(s). Default: (none)
-out
Output beta file. Default: (none)
-design
File containing design matrix. Required.
-fwhm
FWHM of Gaussian temporal smoothing in seconds. Default: 4
-global
Whether to regress out a global signal. Default: false
-minval
Signal threshold. Default: 0
