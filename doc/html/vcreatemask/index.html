vcreatemask
Usage

vcreatemask is a tiny script which creates a mask using a set of vista functional images as input. The script is actually considered to be applied on your preprocessed data (e.g. coming from vpreproc). The created mask then can be used with various lipsia programms which need a mask for defining a certain ROI (e.g. vecm). The standard call looks like this:

> vcreatemask --in directory --out mask.v --maxvoxel 60000

where 'directory' denotes the directory containing all your functional vista images. The parameter 'maxvoxel' denotes the maximal amount of voxel of the mask. If you omit this parameter it will be set to the default value of 50000.

Checking and modifying the mask

After the mask is created a window will pop up presenting you the result. If the mask does not meet your requirements you can edit it by using 'vledit'. To do so type:

> vledit -in mask_reference.v

In the window 'segments' use 'open segments' and load the even created mask. If the mask looks ok to you the file mask_reference.v can be deleted.
