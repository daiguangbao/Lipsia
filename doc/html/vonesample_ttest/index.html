LIPSIA     Random effects analysis - statistical significance of an effect
vonesample_ttest
This program performs a one-sample t-test on a group of contrast images. The ouput is a tmap or a zmap that gives the voxel-wise statistical significance of an effect.

vonesample_ttest -in cgtbsm_*.v -out zmap.v

Caution: The program 'vonesample_ttest' should only be used on registered data, i.e. on data that are geometrically aligned and normalized.

Parameters of 'vonesample_ttest':
-help
Prints usage information.
-in
Input files. Default: (none)
-type [ tmap | zmap ]
Type of output. Default: zmap
-out
Output file. Default: (none)
