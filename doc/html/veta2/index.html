veta2

Background
The program 'veta2' computes a similarity matrix based on functional connectivity between fMRI time series. This matrix should subsequently be analyzed using other programs, e.g. vspectralcluster.

Reference:
Cohen A.L., Fair D.A., Dosenbach N.U., Miezin F.M., Dierker D., Van Essen D.C., Schlaggar B.L., Petersen S.E. (2008). Defining functional areas in individual human brains using resting functional connectivity MRI. Neuroimage 41(1):45-57.

Usage
Here is a typical calling sequence which will produce a clustering of fMRI data within a specified region of interest with 3 clusters. The region of interest can be defined using 'vledit' or some other means.



veta2 -in data.v -out mat.v -mask roi.v

vspectralcluster -in mat.v -out result.v -n 3


Parameters of 'veta2':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-mask
Mask defining a region of interest. Required.
-minval
Signal threshold. Default: 0
-first
First timestep to use. Default: 2
-length
length of time series to use, '0' to use full length. Default: 0
