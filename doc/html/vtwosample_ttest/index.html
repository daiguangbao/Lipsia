LIPSIA     Statistical comparison of two groups
vtwosample_ttest
This program performs a two-sample t-test between two groups of contrast images. The ouput is a tmap or a zmap that gives the statistical significance of a voxel-wise difference between the two groups.

vtwosample_ttest -in1 cgtbsm_group1_*.v -in2 cgtbsm_group2_*.v -out zmap.v

Positive values in the output map indicate that the mean values in group 1 ('-in1') are larger than mean values in group 2 ('-in2'). In line with the LIPSIA convention when determining contrasts, negative values indicate that mean group 1 < mean group 2.

Caution: The program 'vtwosample_ttest' should only be used on registered data, i.e. on data that are geometrically aligned and normalized.

Parameters of 'vtwosample_ttest':
-help
Prints usage information.
-in1
Input files of 1st group. Default: (none)
-in2
Input files of 2nd group. Default: (none)
-type [ tmap | zmap ]
Type of output. Default: zmap
-out
Output file. Default: (none)
