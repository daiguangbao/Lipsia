LIPSIA     Detrending
vdetrend

Background
The program 'vdetrend' removes baseline drifts from functional MRI data by subtracting a moving average.

Usage
In the following example, a moving average of length 60 seconds is subtracted from the data.

vdetrend -in data.v -out data_detrended.v -window 60



Parameters of 'vdetrend':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-window
Size of moving average window in seconds. Default: 60
