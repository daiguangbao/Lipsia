LIPSIA     Geometric distortion correction using a field-map scan
vdistortionCorrection
This program performs a distortion correction using a field-map scan. The following command can be used for the application of the fieldmap 'fieldmap.v' to the functional dataset 'data.v':

vdistortionCorrection -in data.v -out dc_data.v -shift fieldmap.v

The fieldmap is given by the parameter 'shift'. Two other options are available with names 'sigma' and 'edist'. Both are applied only for the preprocessing of the acquired fieldmap. Sigma defines the degree of smoothing of the fieldmap whereas edist deals with the bones and muscles at the border to the brain.

Parameters of 'vdistortionCorrection':
-help
Prints usage information.
-in
Input files. Default: (none)
-out
Output file. Default: (none)
-sigma
sigma for Gauss filter. Default: 0.7
-edist
edist. Default: 3
-shift
Shift image. Required.
