LIPSIA     Power spectrum analysis
vpowerspectrum

Background
The program 'vpowerspectrum' calculates the power spectrum of fMRI data, and produces a map showing several characteristics of the power spectrum at each voxel.


Usage
The program 'vpowerspectrum' calculates various characteristics of the power spectrum which can be select using the parameter '-type'. If '-type range' is selected, the relative power within a given frequency range is computed. The range is given by the parameters '-low' and '-high' which give the wavelengths of the range in seconds. For example, the following call computes the power between 1/10 Hz and 1/30 Hz and relates it to the overall power.

vpowerspectrum -in data.v -out result.v -type range -low 10 -high 30



Alternatively, the spectral centroid can be computed by specifying '-type centroid'. The spectral centroid indicates where the "center of mass" of the spectrum is. It is calculated as the weighted mean of the frequencies present in the signal with their magnitudes as the weights. The parameters '-low' and '-high' may be used to determine a range in which the centroid is calculated. If left to the default value of -1, the entire spectral range is used. Analogously, the spectral median may be calculated using '-type median'. Example:

vpowerspectrum -in data.v -out result.v -type centroid



Parameters of 'vpowerspectrum':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-tr
Repetition time in seconds. Default: -1
If tr=-1, the repetition time specified in the image header is used.
-type [range | median | centroid | overall]
Type of output. Default: range
-low
Lower bound of frequency range in seconds.
Default: -1
-high
Upper bound of frequency range in seconds.
Default: -1
-minval
Signal threshold to exclude non-brain voxels. Default: 0
