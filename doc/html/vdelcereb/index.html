LIPSIA     Remove cerebellum from a white matter segmentation
vdelcereb
The program 'vdelcereb' is used to remove the cerebellum from a white matter segmentation. The input into 'vdelcereb' must be a white matter segmentation obtained by 'vwhitematter'.:

Here is an example:

vwhitematter -in infile.v -out wmfile.v

vdelcereb -in wmfile.v -out outfile.v

The input file to the program 'vwhitematter' should be a T1-weighted, high-resolution image (e.g. from the MPI inhouse data base /home/newbdb/024/mr024_t1_pl.v). The output of 'vwhitematter' is a white matter segmentation. It may be used as input into 'vdelcereb'. 'vdelcereb' deletes the cerebellum from the white matter segmentation as shown below.



Parameters of 'vdelcereb':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-verbose
Show program messages. Default: 0
