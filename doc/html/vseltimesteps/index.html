LIPSIA     Selection of timesteps
vseltimesteps
The program 'vseltimesteps' is used to select timesteps from fMRI data. 'vseltimesteps' is similar to the program 'vselbands' (see 'via-help' for detailed information). However, 'vselbands' does not distinguish between anatomical data and functional data. Thus, 'vselbands' selects slices not only from the functional data but also from the anatomical data. 'vseltimesteps' ignores the anatomical data.

Here is an example:

vseltimesteps -in infile.v -out outfile.v -step 0 to 234

This call selects all timesteps from 0 to 234 and writes them into the output file. It leaves the anatomical data unchanged.

Parameters of 'vseltimesteps':
-help
Prints usage information.
-in
Input file. Default: (none)
-out
Output file. Default: (none)
-step
Timesteps to be selected (counting begins at 0). Default: (none)
